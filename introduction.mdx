<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold dark:text-white text-gray-900 mb-4">
    Quick start
  </h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
    Get up and running with the monorepo in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Start by cloning the FE Monorepo template to your local machine:

      ```bash
      git clone https://github.com/rifandani/fe-monorepo.git
      cd fe-monorepo
      ```
    </Step>

    <Step title="Install dependencies">
      The monorepo uses Bun as the package manager. Install all dependencies:

      <CodeGroup>
      ```bash Bun
      bun install
      ```

      ```bash npm
      npm install
      ```

      ```bash yarn
      yarn install
      ```
      </CodeGroup>

      <Note>
        This project requires Node.js >= 24.14.0 and uses Bun 1.3.10 as specified in `package.json`.
      </Note>
    </Step>

    <Step title="Run a development server">
      Choose which app you want to run:

      <Tabs>
        <Tab title="SPA (React)">
          Start the React 19 SPA on port 3001:
          ```bash
          bun spa dev
          ```
        </Tab>
        <Tab title="Web (Next.js)">
          Start the Next.js 16 app on port 3002:
          ```bash
          bun web dev
          ```
        </Tab>
        <Tab title="Expo (Mobile)">
          Start the Expo 53 development server:
          ```bash
          bun expo dev
          ```
        </Tab>
      </Tabs>
    </Step>

    <Step title="Start building">
      Open your browser and start building your application. Each app comes pre-configured with:
      
      - TypeScript with strict type checking
      - ESLint for code quality
      - Playwright/Maestro for E2E testing
      - OpenTelemetry for observability
      - Hot module replacement for fast development

      <Accordion title="View example project structure">
        ```
        fe-monorepo/
        ├── apps/
        │   ├── spa/          # React 19 + TanStack Router
        │   ├── web/          # Next.js 16 + Better Auth
        │   └── expo/         # Expo 53 + Tamagui
        ├── packages/
        │   ├── core/         # Shared business logic
        │   └── typescript-config/
        └── package.json
        ```
      </Accordion>
    </Step>
  </Steps>
</div>
